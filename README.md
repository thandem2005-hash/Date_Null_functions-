# SQL Date & NULL Functions – Practise Exercises

This project demonstrates the use of advanced SQL functions focusing on DATE and NULL handling. It includes practical queries that manipulate dates (e.g., DATEADD, DATEDIFF, TO_CHAR, EXTRACT) and manage missing values using functions like COALESCE, ISNULL, and IFNULL.

The DATE functions section covers operations such as calculating time differences, formatting dates, extracting date parts, and adjusting timelines. The NULL functions section focuses on handling incomplete data by replacing, counting, and evaluating NULL values to ensure accurate analysis.

Each exercise includes SQL queries and expected outputs, helping reinforce real-world data cleaning and transformation skills.

This project is ideal for building a strong foundation in SQL for data analytics and preparing datasets for reporting tools like Power BI.
